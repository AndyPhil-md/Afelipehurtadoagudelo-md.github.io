# Afelipehurtadoagudelo-md.github.io

Diario personal hecho con **Hugo**.

Este repositorio contiene mi sitio web personal: un diario en línea donde voy dejando apuntes, ensayos, cartas, notas de lectura y fragmentos de pensamiento. No está pensado como una vitrina ni como una marca personal, sino como un archivo vivo: una forma de rescatar ideas, fijar memoria y dejar rastro de procesos, obsesiones y preguntas.

## Qué es este sitio

Este sitio funciona como un **diario público**.

Aquí conviven textos de distintos registros:

- entradas más ensayísticas
- notas breves
- borradores con vocación de ensayo
- cartas
- apuntes de lectura
- fragmentos de un *commonplace book*

No todo busca cierre. A veces el valor del texto está en dejar constancia, no en concluir.

## Idea del proyecto

La idea no es producir “contenido”, sino sostener un lugar donde pensar por escrito.

Me interesa usar la web como:

- memoria
- cuaderno
- archivo
- laboratorio de ideas
- rastro personal del presente

## Estructura general

La mayor parte del contenido está dentro de `content/post/`.

Ahí conviven textos sobre:

- educación
- poder
- cultura
- vida cotidiana
- consumo
- lecturas
- experiencia personal

## Stack

- **Static site generator:** Hugo
- **Theme:** `hugo-theme-stack`
- **Deploy:** GitHub Pages

## Publicación

El flujo es simple:

1. escribir un nuevo `.md`
2. hacer commit
3. hacer push
4. GitHub Pages recompila y publica

## Nota editorial

Este repositorio no intenta simular una obra terminada.
Es un diario.
Por eso hay textos más pulidos y otros más fragmentarios.
Lo importante no es la homogeneidad, sino la continuidad.
